Тестовое задание для студентов
Цель: Практика в использовании командной строки и инструментов разработки для взаимодействия с REST API через curl.

Описание задания:

Студентам предлагается выполнить серию запросов к REST API бэкенд-приложения для списка задач, используя инструмент curl. Задачи будут включать создание, чтение, обновление и удаление задач.

Задачи

Получить список всех задач.
Используйте curl для отправки GET-запроса к API, чтобы получить список всех задач.

Добавить новую задачу.
Используйте curl для отправки POST-запроса к API с телом запроса, содержащим название новой задачи.

Получить одну задачу по ID.
Используйте curl для отправки GET-запроса к API, чтобы получить информацию о задаче с конкретным ID.

Обновить задачу.
Используйте curl для отправки PUT-запроса к API с телом запроса, содержащим новое название задачи.

Удалить задачу.
Используйте curl для отправки DELETE-запроса к API, чтобы удалить задачу с конкретным ID.

Инструкции
Для каждого задания предоставьте соответствующую команду curl, которую необходимо выполнить для взаимодействия с API. Убедитесь, что вы используете правильный URL и формат данных, если это необходимо.
